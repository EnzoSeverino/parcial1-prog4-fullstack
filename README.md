# Primer Parcial Integrador - Programación IV 🚀

Aplicación Fullstack desarrollada para la gestión de un catálogo de productos, categorías e ingredientes. 

## Tecnologías Utilizadas
* **Backend:** Python, FastAPI, SQLModel, PostgreSQL.
* **Frontend:** React, TypeScript, Vite, Tailwind CSS, TanStack Query, React Router DOM.
* **Patrones:** Arquitectura en capas, Unit of Work.

## Instalación y Ejecución
### Backend
1. Crear entorno virtual: `python -m venv venv`
2. Activar entorno virtual: `source venv/Scripts/activate` (o `venv\Scripts\activate` en Windows)
3. Instalar dependencias: `pip install -r requirements.txt`
4. Ejecutar servidor: `fastapi dev main.py`

### Frontend
1. Instalar dependencias: `pnpm install`
2. Ejecutar servidor de desarrollo: `pnpm dev`
